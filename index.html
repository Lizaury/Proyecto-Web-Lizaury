¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para estructurar y presentar contenido en la web. HTML es esencial para la creación de sitios web, permitiendo que el contenido sea legible tanto para usuarios como para navegadores. HTML5 es la última versión del lenguaje de marcado estándar para la web, que mejora la semántica, la accesibilidad y la compatibilidad con tecnologías modernas. Introduce nuevas etiquetas como <article>, <section< y <header>, y soporta de manera nativa audio, video y gráficos sin necesidad de plugins externos. Además, ofrece funciones avanzadas como almacenamiento local, geolocalización y soporte para aplicaciones web sin conexión, lo que facilita la creación de sitios interactivos y optimizados para dispositivos móviles y navegadores modernos. HTML5 es ahora el estándar principal para el desarrollo web.

Historia HTML

HTML (HyperText Markup Language) fue creado en 1991 por Tim Berners-Lee como un estándar para estructurar y enlazar documentos en la web. En sus inicios, era una herramienta básica para compartir información científica, pero rápidamente evolucionó con la llegada de nuevos navegadores que añadieron más capacidades. En 1995, se estandarizó con HTML 2.0, y versiones posteriores, como HTML 4.01 en 1999, mejoraron su funcionalidad. En 2014, HTML5 revolucionó el lenguaje, añadiendo soporte para multimedia y una estructura más semántica, adaptándose a la creciente demanda de dispositivos móviles y aplicaciones modernas. Actualmente, HTML sigue evolucionando para satisfacer las necesidades de una web dinámica y en constante crecimiento.

Editores HTML

Los editores de HTML se clasifican en simples, avanzados, visuales y en línea, y se adaptan a diferentes niveles de experiencia. Los simples, como Notepad y TextEdit, son ideales para principiantes. Los avanzados ofrecen herramientas potentes como resaltado de sintaxis y extensiones, mientras que los visuales combinan diseño y código. Los editores en línea permiten probar y compartir código en tiempo real.

  • Editores populares:

  • Simples: Notepad, TextEdit.
  • Avanzados: Visual Studio Code, Sublime Text, Atom.
  • Visuales: Adobe Dreamweaver, BlueGriffon.
  • En línea: CodePen, JSFiddle.

Tu primer documento HTML

Para crear un documento HTML, primero abre un editor de texto simple como Notepad (en Windows) o TextEdit (en Mac) y guarda el archivo con la extensión .html. El contenido básico de un archivo HTML comienza con la declaración <!DOCTYPE html> que indica al navegador que es un documento HTML5. Luego, se escribe la etiqueta <html> que engloba todo el contenido del documento. Dentro de la etiqueta <html>, se añaden dos secciones: <head>, donde se colocan metadatos como el título del con la etiqueta <title>, y <body>, que es donde se coloca el contenido visible, como texto y imágenes, utilizando etiquetas como <h1> para encabezados y <p> para párrafos. Al finalizar, guarda el archivo y abrelo en un navegador para ver cómo se muestra.

¿Que son Etiquetas en HTML?

Las etiquetas en HTML son elementos básicos que se utilizan para estructurar y definir el contenido de una página web. Están delimitadas por los signos <> y suelen venir en pares: una etiqueta de apertura, como <p>, y una de cierre, como </p>. Entre estas etiquetas se escribe el contenido que se desea mostrar o estilizar. Cada etiqueta tiene una función específica; por ejemplo, <h1> define un encabezado, <p> un párrafo y <a> un enlace. Algunas etiquetas, como <img> para imágenes o <br> para saltos de línea, son de una sola pieza y no necesitan cierre. Estas etiquetas permiten organizar, dar estilo y añadir interactividad al contenido web.

¿Que son Atributos en HTML?

Los atributos en HTML son propiedades adicionales que se añaden a las etiquetas para proporcionar más información o personalizar su comportamiento. Se escriben dentro de la etiqueta de apertura, en formato de pares nombre="valor". Por ejemplo, en la etiqueta <a href="https://example.com">, el atributo href especifica la URL del enlace. Otros atributos comunes incluyen id, que asigna un identificador único; class, que permite agrupar elementos para aplicar estilos o scripts; y src, usado en etiquetas como <img> para indicar la ubicación de una imagen. Los atributos ayudan a controlar el aspecto, la funcionalidad y la interacción de los elementos en la página web.

Reglas fundamentales en HTML

Las reglas fundamentales de HTML incluyen comenzar con <!DOCTYPE html> y estructurar el contenido dentro de las etiquetas <html>, <head> y <body>. Las etiquetas deben abrirse y cerrarse correctamente, salvo las auto-contenidas como <img>. Es recomendable usar nombres y atributos en minúsculas, valores de atributos entre comillas y declarar la codificación de caracteres con <meta charset="UTF-8">. Además, utiliza etiquetas semánticas modernas para mejorar la accesibilidad y evita las obsoletas como <font>. Mantén el código limpio con indentación y comentarios, y valida el documento con herramientas como el W3C Validator para garantizar su calidad.

Etiquetas Para hacer un documento HTML

Durante la creación de un documento HTML se utilizan una serie de etiquetas que puedan organizar y estructurar el documento de manera correcta.

Etiqueta Descripción
<!DOCTYPE html> Declara el tipo de documento como HTML5
<html> Contiene todo el contenido de la página web
<head> Incluye información del documento como el título y metadatos
<title> Define el título que aparece en la pestaña del navegador
<meta> Proporciona metadatos como la codificación de caracteres (<meta charset="UTF-8">)
<body> Contiene el contenido visible de la página
<header> Define la cabecera del documento o sección
<main> Contiene el contenido principal del documento
<section> Agrupa contenido relacionado dentro del documento
<article> Define contenido independiente, como un artículo o post
<footer> Representa el pie de página con información adicional
<h1> a <h6> Establecen los encabezados, de mayor a menor importancia
<p> Define párrafos para el texto
<ul> y <ol> Crean listas desordenadas y ordenadas respectivamente
<table> Estructura datos tabulares; y <blockquote> indica citas largas
